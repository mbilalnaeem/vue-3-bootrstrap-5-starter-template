# Vue 3 Bootstrap 5 Starter Template with Ability to customize and overrides default bootstrap sass

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
